# 系统操作说明

介绍系统基本的页面结构与配置操作。 用户可以通过运输地点、车型、承运商、承运商可用资源、商品大类信息的维护，有效管理运单规划所需要使用的主数据情况。

# 1. 登陆

浏览器输入小马驾驾网址 t.d2d.ai, 填入用户名/密码，点击“登陆”，进入系统。

# 2. 列表页面功能概览

# 2.1 页面分区介绍

登陆之后页面分为顶部信息区，左边菜单区域，右边功能区域。

  • 功能区域分为二级菜单，点击一级菜单可展开相应的二级菜单功能。
  • 点击二级菜单可以打开具体的功能页面，默认展开相应数据对象的列表记录，超链接可点击查看。

# 2.2 页面操作功能介绍

  • 列表页面下方是功能按钮区域，列表页面上方是快捷查询区域
  • 部分列表页面支持模版查询，用户可预设相关的查询模版
  • 当前页面支持2000行/页的流畅操作

# 2.3 页面查询功能介绍

  • 点击“更多”展开全部查询条件列表
  • 点击“表格配置”展开表格配置页面

# 3. 详情页面功能概览

# 3.1 详情页面进入方式

  • 点击编辑或者列表页面的对象主键超链接（例如运单列表内点击运单编号），跳转详情页面。

# 3.2 详情页面概览

  • 多Tab形式的详情页面，隶属于同一个主要对象，点击tab名称切换.

# 4. 查询页面功能概览

  • 详情页面支持模版查询的方式。
  • 通过页面顶部的模版输入框选择模版。
  • 用户可以将查询搜索条件保存为查询模版共后续使用。
  • 点击“更多”，输入查询条件后保存模版。
