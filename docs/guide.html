# 新手教程

指导新手用户使用小马驾驾运输优化系统完成一次基本的优化和调度。

# 第1步：登陆系统

  • 打开浏览器（浏览器不限，推荐Chrome浏览器），输入网址 t.d2d.ai，进入小马驾驾运输优化系统。
  • 输入登陆账户、密码，登陆系统。

# 第2步：基本数据准备

  • 发起一次基础功能的优化前需要维护本次优化所需的数据。
  • 数据包括运单、运输地点、商品、车型、承运商、承运商可用资源、优化参数等。
  • 可以通过系统页面手工新建与调整数据，也可以通过Excel批量上传。
  • 此处介绍通过模版上传进行配置的方法。

# 1. 批量上传

  • 系统的批量上传入口为：运输计划>上传请求；

  • 在该菜单页点击 “模版下载” 可以按照需要下载相应的模版。

    • “运单”对应 “运单新增模版”(通过模版批量上传的方式上传“运单新增模版”可以同时配置运输地点和运单信息)
    • “车型”对应 “车型新增模版”
    • “承运商可用资源” 对应 “承运商可用资源模版”(上传前需要先进入页面 “基础资料>承运商” 新增承运商，再通过该模版为承运商添加可用资源)

  • 模版为Excel(.xlsx)文件，按照规定的规则和要求填写模版内容后，点击 “文件上传” 按钮，选择该填好的模版上传即可。

  • 待系统处理好可以点击上传请求ID超链接或者勾选本次请求点击“查看详情”查看本次请求的结果，如有失败的数据，将会指明数据在表格中的位置和失败原因。

# 2.优化参数配置

  • 系统支持用户自定义配置参数集，配置入口为：运输计划>优化参数；
  • 输入优化参数集描述，选择优化器版本（推荐“标准版算法”），然后配置相应的参数。
  • 新手初次使用，在没有特殊的优化需求时，采用参数默认值即可，可以保证良好快速的优化优化性能。

# 第3步：发起优化

在配置好上述参数与数据后，可以发起一次优化，发起入口：“运输计划>运单”。

  • 通过勾选框☑️选择需要优化的运单，
  • 点击页面下方 “优化” 按钮，
  • 在弹窗中选择本次优化的参数集，
  • 点击弹窗中的 “优化” 按钮，即可发起本次优化。

# 第4步：查看结果

发起优化后，数分钟即可完成优化。

  • 可以通过 “运输计划>优化请求” 查看本次优化的进程与状态。
  • 优化完的结果以车次的形式存在，可以通过 “运输计划>车次” 查看算法输出的车次结果，该页面页可以进行地图展示与调整、车次下载等，如需了解车次详情，点击车次号超链接进入详情页。
  • 查看好并确认车次可行后，在 “运输计划>车次” 中勾选确认的车次，点击页面下方的 “确认车次” 按钮，完成一次完整的优化流程。
