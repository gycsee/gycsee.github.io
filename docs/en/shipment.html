车次

车次状态修改接口文档说明

  1. 请求链接
  • /api/vrp/v1/shipment/updatestate
  1. 请求类型
  • PATCH
  1. 请求参数
请求参数参数类型参数说明
shipmentIdsList<String>待修改状态的车次ID列表
shipmentStateChangeFlagint车次状态修改标志
  1. 请求参数示例
{
  "shipmentIds": [
    "389",
    "390"
  ], 
  "shipmentStateChangeFlag": 1
}
  1. 返回参数
返回参数参数类型参数说明
httpCodeIntegerhttp响应编码
retCodeInteger请求结果编码
retInfoString英文信息
retMsgString中文信息
dataObject返回数据实体
data.errorObject[]错误信息列表
data.error[].errorString错误信息详情
data.error[].idString处理失败的车次ID
data.successString[]处理成功的车次ID列表
  1. 返回参数示例
{
    "data": {
        "error": [
        {
            "error": "车次已经是目标状态",
            "id": "389"
        }
        ],
        "success": [
        "390"
        ]
    },
    "httpCode": 200,
    "retCode": 0,
    "retInfo": "OPERATE_SUCESS",
    "retMsg": "操作成功"
}
