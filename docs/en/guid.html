新增文档教程

1. 编辑 markdown 文档

在根目录下docs文件夹下新建文件（如example.md）,文件开头增加如下代码

---
id: guid
title: 新增文档教程
sidebar_label: 新增文档教程
---

其中 id 为文档唯一标识，docs 下所有文档不能重叠，已经命名的 id 可在 /website/sidebars.json 中查看，title 为文档页html的标题，sidebar_label 为文档在左边栏的标题，如过 sidebar_label 为空，则使用 title，特别注意， markdown 文档中的二级标题会自动在右侧生成文档目录

2. 将文档加入到左边栏中

修改 /website/sidebars.json ，将文档的 id 添加到 docs 中指定分类下，可增加分类

3. 测试

打开命令行工具，进入项目 /website/ 文件夹下，执行命令

npm run start

无报错情况下会开启本地3000端口的服务器，在浏览器中输入 http://localhost:3000 即可以查看刚才编写的文档

4. 测试通过则提交代码
