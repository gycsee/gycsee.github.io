费率区域

1 根据查询条件获取区域列表

1.1 请求链接

  • <host>/api/vrp/v1/zone/queryZone

1.2 请求类型

  • POST

1.3 请求参数

请求参数参数类型参数说明是否必填
pageInnteger页码是
sizeInteger每页展示条数是
zoneIdString区域ID否
zoneDescString区域描述否
layerStringzone所属图层否
deptIdString部门ID否
branchIdString公司ID否
createDtt.startDateLocalDate开始创建时间否
createDtt.endDateLocalDate截止创建时间否
createByString创建人否
updateDtt.startDateLocalDate开始更新时间否
updateDtt.endDttLocalDate截止更新时间否
updateByString最后更新人否

1.4 请求参数示例

{
  page: 1, 
  size: 10, 
  zoneId: '',
  zoneDesc: '',
  layer: '',
  deptId: '',
  branchId: '',
  createDtt: {
    endDate: '',
    startDate: ''
  },
  createBy: '',
  updateDtt: {
  endDate: '',
    startDate: ''
  },
  updateBy: ''
}

1.5 返回结果

返回参数参数类型参数描述
httpCodeIntegerhttp状态码
retCodeInnteger返回结果编码
retInfoString返回结果英文信息
retMsgString返回结果中文信息
zoneIdString区域ID
zoneDescString区域描述
layerString所属图层
deptIdString公司ID
deptNameString公司名称
branchIdString部门ID
branchNameString部门名称
createDttLocalDateTime创建时间（yyyy-mm-dd hh:mm:ss）
createByString创建人
updateDttLocalDateTime最后更新时间（yyyy-mm-dd hh:mm:ss）
updateByString最后更新人

1.6 返回结果示例

{
list: [{
    zoneId: 'aaa',

5 批量删除区域

5.1 请求链接

  • <host>/api/vrp/v1/zone/deleteBatchZone

5.2 请求类型

  • DELETE

5.3 请求参数

请求参数参数类型参数说明是否必填
zoneIdsList<String>区域ID列表是

5.4 请求参数示例

{ zoneIds: ['zone_id_1', 'zone_id_2', 'zone_id_3'] }

5.5 返回结果

返回参数参数类型参数说明
httpCodeIntegerhttp状态码
retCodeInnteger返回结果编码
retInfoString返回结果英文信息
retMsgString返回结果中文信息
errorList<ErrorVO>删除失败信息列表
successList<String>删除成功的id列表

5.6 返回结果示例

{
  success: ['zone_id_1'],
  error: [{
    id: 'zone_id_2',
    error: 'zone_id_2'
    }, {
    id: 'zone_id_3',
    error: 'zone_id_3',
    }
  ]
}
