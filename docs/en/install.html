项目安装向导

环境要求

  1. 安装 node
  2. 安装 git

安装

  1. 下载或者 clone 项目

    文件结构

     ```
 root-directory
   ├── docs
   │   ├── doc1.md
   │   ├── doc2.md
   │   └── doc3.md
   └── website
       ├── blog
       │   ├── 2016-03-11-blog-post.md
       │   ├── 2017-04-10-blog-post-two.md
       │   ├── 2017-09-25-testing-rss.md
       │   ├── 2017-09-26-adding-rss.md
       │   └── 2017-10-24-new-version-1.0.0.md
       ├── core
       │   └── Footer.js
       ├── package.json
       ├── pages
       ├── sidebars.json
       ├── siteConfig.js
       └── static                        
 ```

  2. 命令行进入项目文件夹

    cd website
npm install

  3. 测试

    npm run start

    如果成功开启本地3000端口的web服务则安装成功（浏览器中输入http://localhost:3000，一般会自动跳转）
