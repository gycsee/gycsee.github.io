根据上传请求ID列表删除

  1. 请求URL

<host>/api/vrp/v1/integrations

  1. 请求方式
  • DETELE
  1. 请求参数
请求参数参数类型参数说明
idsList<Integer>,可为空上传ID列表
  1. 请求示例
{
    ids: [
      1,2,3
    ]
}
  1. 返回参数
返回参数参数类型参数说明
httpCodeIntegerResoponse Code 标准返回码 (500,200,401,403,404)
retCodeInteger执行结果code
retMsgString状态的英文代码
retInfoString状态描述
dataObject数据
  1. 返回示例
{
  httpCode: 500,
  reCode: 10031000,
  retMsg: "SYS_ERR",
  retInfo: "系统错误"，
  data:{
    success: [
            1,2,3,4
        ],
        error: [
          {
            id:5,
            error: "error-info"
          }  
        ]
  } 
}

下载错误文件

下载错误文件。当下载失败时，返回错误信息。下载成功时，返回数据流

  1. 请求URL

<host>/api/vrp/v1/integration/{id}/error-file

  1. 请求方式
  • GET
  1. 请求参数
请求参数参数类型参数说明
idInteger，不可为空上传请求ID
  1. 返回
返回参数参数类型参数说明
httpCodeIntegerResoponse Code 标准返回码 (500,200,401,403,404)
retCodeInteger执行结果code
retMsgString状态的英文代码
retInfoString状态描述
  1. 返回示例
{
  httpCode: 500,
  reCode: 10031203,
  retMsg: "INTEGRATION_REQ_ID_NOT_EXIST",
  retInfo: "上传请求ID不存在"
}
