优化请求

优化请求日志获取接口文档

根据优化请求ID和优化请求日志类型

  1. 请求URL
  • <host>/api/vrp/v1/optimization/{id}/logs
  1. 请求方式
  • GET
  1. 请求参数
请求参数参数类型参数说明
idString,不可为空优化请求ID
typeString日志类型
  1. 请求示例
* <host>/api/vrp/v1/optimization/20180504174427/logs?type=OPT
  1. 返回参数
返回参数参数类型参数说明
httpCodeIntegerResoponse Code 标准返回码 (500,200,401,403,404)
retCodeInteger执行结果code
retMsgString状态的英文代码
retInfoString状态描述
dataObject数据
executeDateString执行日期
stepInterge步骤
actionDescString动作描述
keyValue1String关键值1
  1. 返回示例
{
  httpCode: 500,
  reCode: 10031000,
  retMsg: "SYS_ERR",
  retInfo: "系统错误"，
  data:[
    {
        actionDesc: "开始创建优化请求,请求ID:{0}",
        executeDate: "2018-05-04 17:44:27",
        keyValue1: "20180504174427",
        step: 1001
      }
  ]
}
