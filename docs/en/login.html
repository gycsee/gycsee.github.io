登陆接口

  1. 请求链接
  • <host>/permission/login
  1. 请求类型
  • POST

  1. 请求参数 | 请求参数 | 参数类型 | 参数说明| 是否必填 | |:-:|:-:|:-:|:-:| | username | String | 用户名 | 是 | | password | String | 密码 | 是 |

  2. 请求参数示例 { "password": "123456", "username": "yuguanquan@shanshu.ai" }

  3. 返回结果 | 返回参数 | 参数类型 | 参数说明 | |:-:|:-:|:-:| | httpCode | Integer | http状态码 | | retCode | Integer | 返回结果编码 | | retInfo | String | 返回结果英文信息 | | retMsg | String | 返回结果中文信息 | | id | Long | 主键ID | | userCode | String | 用户编码 | | userName | String | 用户名 | | passwordState | String | 密码状态: INIT 初始密码， NORMAL 非初始密码 | 否 | | token | String | 用户权限token | | surname | String | 姓氏 | | givenname | String | 名字 | | nickname | String | 昵称 | | avatar | String | 头像 | | birthday | Date | 出生日期(yyyy-MM-dd) | | gender | Boolean | 性别, false为女士，true位男士 | | email | String | 电子邮箱 | | mobile | String | 手机号码 | | address | String | 地址 | | zipCode | String | 邮编 | | joinTime | LocalDateTime | 入职时间(yyyy-MM-dd hh:mm:ss) | | comments | String | 描述信息 | | active | Boolean | 是否启用(true:启用;false:不启用) | | createTime | LocalDateTime | 创建时间(yyyy-MM-dd hh:mm:ss) | | updateTime | LocalDateTime | 更新时间(yyyy-MM-dd hh:mm:ss) |
