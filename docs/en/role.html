添加角色接口

  1. 请求链接
  • <host>/permission/role
  1. 请求类型
  • POST
  1. 请求参数
请求参数参数类型参数说明是否必填
projectString项目名称(Header)是
moduleString模块名称(Header)是
roleNameString角色名是
deptCodeString部门编码是
commentsString描述信息否
privilegedResourceCodesList<String>权限资源编码列表是
activeBoolean是否启用(true:启用;false:不启用)是
  1. 请求参数示例
project=unicorn
module=backend
{
    "active": true,
    "comments": "管理员",
    "deptCode": "b093a2bfb0f548e68cfe93f703a19121",
    "roleName": "管理员"
}
  1. 返回结果
返回参数参数类型参数说明
httpCodeIntegerhttp状态码
retCodeInteger返回结果编码

retMsgString返回结果中文信息

roleNameString角色名称

privilegedInfoVOString权限资源
button/menu.privilegedBoolean是否授权
button/menu.resourceCodeString权限资源编码
button/menu.resourceNameString权限资源名称
button/menu.parentResourceCodeString权限资源父节点编码
button/menu.privilegedBoolean是否授权
menu.childrenList子菜单列表

createTimeLocalDateTime创建时间(yyyy-MM-dd hh:mm:ss)
updateTimeLocalDateTime更新时间(yyyy-MM-dd hh:mm:ss)

更新角色接口

  1. 请求链接
  • <host>/permission/role
  1. 请求类型
  • PATCH
  1. 请求参数
请求参数参数类型参数说明是否必填
projectString项目名称(Header)是
moduleString模块名称(Header)是
roleCodeString角色编码是
roleNameString角色名称否
deptCodeString部门编码否
commentsString描述信息否
revokePrivilegedResourceCodesList<String>取消授权资源列表否
privilegedResourceCodesList<String>授权资源列表否
activeBoolean是否启用(true:启用;false:不启用)否
  1. 请求参数示例
project=unicorn
module=backend
{
    "active": true,
    "comments": "测试管理员",
    "roleName":"测试管理员",
    "deptCode":"b093a2bfb0f548e68cfe93f703a19121",
    "roleCode": "356200dcd49f4f8b8421214b8bd014c5"
    "revokePrivilegedResourceCodes":[
        "356200dcd49f4f8b8421214b8bd01434",
        "246200dcd49f4f8b8421214b8bd01435"
    ],
    "privilegedResourceCodes":[
        "356200dcd49f4f8b8421214b8bd01436",
        "246200dcd49f4f8b8421214b8bd01437"
    ]
}

根据角色编码列表批量删除角色接口

  • <host>/permission/role/codes
  1. 请求类型
  • DELETE
  1. 请求参数
请求参数参数类型参数说明是否必填
dataList<String>角色编码列表 是
  1. 请求参数示例
{
    "data": [
        "edfc7b66d01d4189afe7cb8b95d8403e",
        "a437b2d8f7a642ce85beaa4d9a573db4"
    ]
}
  1. 返回结果
返回参数参数类型参数说明
httpCodeIntegerhttp状态码
retCodeInteger返回结果编码
retInfoString返回结果英文信息
retMsgString返回结果中文信息
successCodesList<String>成功删除角色编码列表
errorVOListList<ErrorVO>删除失败信息列表
codeString失败角色编码
errorMsgString删除失败错误信息

{

根据条件查询角色信息接口

  1. 请求链接
  • <host>/permission/role/query
  1. 请求类型
  • POST
  1. 请求参数
请求参数参数类型参数说明是否必填
roleCodeString角色编码否
roleNameString角色名否
commentsString描述信息否
activeBoolean是否启用(true:启用;false:不启用)否
createTime.startDateTimeLocalDateTime开始创建时间(yyyy-MM-dd hh:mm:ss)否
createTime.endDateTimeLocalDateTime截止创建时间(yyyy-MM-dd hh:mm:ss)否
updateTime.startDateTimeLocalDateTime开始更新时间(yyyy-MM-dd hh:mm:ss)否
updateTime.endDateTimeLocalDateTime截止更新时间(yyyy-MM-dd hh:mm:ss)否
pageNoInteger页码是
pageSizeInteger每页条数是

{

    "roleName": "%",
