菜单查询接口

  1. 请求链接
  • <host>/permission/queryMenuAjax
  1. 请求类型
  • GET
  1. 请求参数
请求参数参数类型参数说明是否必填
unicornString项目名称是
backendString模块名称是
  1. 请求参数示例
  • <host>/permission/queryMenuAjax?project=unicorn&module=backend

  1. 返回结果 | 返回参数 | 参数类型 | 参数说明 | |:-:|:-:|:-:| | httpCode | Integer | http状态码 | | retCode | Integer | 返回结果编码 | | retInfo | String | 返回结果英文信息 | | retMsg | String | 返回结果中文信息 | | icon | String | 菜单图标 | | key | String | 菜单key | | link | String | 菜单链接 | | title | String | 菜单标题 | | subMenus | String | 子菜单 |
