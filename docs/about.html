# 产品简介

小马驾驾（PonyPlus）是杉数科技旗下的运输优化系统。通过独有的高效优化算法，为企业提供多维度，多目标，多场景的配送任务分配以及路线规划建议，显著提高企业的运营效率，降低物流决策成本。

# 机遇与挑战

在中国复杂等道路交通与多变的业务需求背景下，传统的包括城市配送、干支线配送等运输场景面临着诸多挑战，例如：

  • 运输资源匹配不合理
  • 时间窗口约束严格
  • 城市分区分车型分时段限行
  • 业务模式复杂多变
  • 手工调度效率低下

如何在考虑多种业务约束条件的同时，全局统筹规划所有资源，在确保满足客户服务水平的前提下，尽可能减少运输成本，提升资源利用率的同时提升业务响应速度，成为了物流效率提升的主要诉求，也是企业在日益激烈的行业竞争中保持领先或力争上游的机遇。

# 业务场景

小马驾驾运输优化系统主要针对城市配送、支干线运输的业务场景，进行包括配送路径优化、装卸时间窗优化、司机排班优化、装箱优化等在内的多重目标优化，实现降低总体运输成本、提升运输计划效率、提升客户满意度、提升业务响应速度等效果。

# 实现原理

经过三年打磨，杉数打造了可解释、可配置、可依靠的算法内核，即杉数运输优化引擎。 根据用户输入的运单、地点、时效、固定线路、车型车辆、费率、商品、装卸效率等信息，可在分钟级时间内优化得到全局最优的排班线路。

# 使命与愿景

小马驾驾的算法与产品设计将紧跟运输行业发展趋势，与时俱进的适应更新更实用的运输要求并扩充应用场景，不断提升优化性能与时效。 我们致力成为智慧物流的大脑，助力企业在日益激烈的行业竞争中保持领先或力争上游。
